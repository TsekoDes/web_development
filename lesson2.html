  • 3 types of buttons

    • A normal button
    • A button used to submit the contents of a form
    • A button that is represented by an image

  • HTML text inputs

    • An input field that accepts text
    • An input field with an obfuscated password
    • An input field meant for paragraphs

  • HTML Hidden Input

    • An input field that is hidden to the user and used to add control data submitted to the server

  • HTML Check Boxes Input
